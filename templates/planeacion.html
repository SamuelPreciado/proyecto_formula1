Objetivo del Proyecto

Desarrollar una API con FastAPI que permita realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre modelos de un campeonato de Fórmula 1: pilotos y carreras. El sistema debe almacenar la información en la base de datos para así garantizar persistencia.
