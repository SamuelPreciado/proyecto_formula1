Mapa de Endpoints

A continuación se detallan los endpoints disponibles en la API:

  • Pilotos:
    • GET /pilotos/ → Lista todos los pilotos
    • POST /pilotos/ → Crea un nuevo piloto
    • GET /pilotos/{id} → Obtiene pilotos por ID
    • PUT /pilotos/{id} → Actualiza un piloto
    • GET /pilotos/{id} → Filtra pilotos por escuderia
    • GET /pilotos/{id} → Obtiene pilotos por nombre
    • DELETE /pilotos/{id} → Elimina un piloto
  • Carreras:
    • GET /carreras/ → Lista todas las carreras
    • POST /carreras/ → Crea una nueva carrera
    • GET /carreras/{id} → Obtiene carrera por ID
    • PUT /carreras/{id} → Actualiza una carrera
    • GET /carreras/{id} → Filtra carreras por pais
    • GET /carreras/{id} → Obtiene carreras por ganador
    • DELETE /carreras/{id} → Elimina una carrera
  • Endpoints - Borrados
    • GET /borrados → Obtiene los pilotos y las carreras borradas
